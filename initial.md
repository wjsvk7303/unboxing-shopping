 프로젝트명 : 쇼핑샾
 프로젝트설명 : 전자기기 제품을 판매하는 온라인 샾 입니다.
 프로젝트목표 : 사용자 친화적인 인터페이스를 제공하여 사용자가 쉽게 제품을 검색하고 구매할 수 있도록 한다.
 
 프로젝트 목적 : tosspayments 이용해서 결제 시스템을 적용해보고 테스트 결제하는 것 까지 전체적인 플로루를 구현 하는 쇼핑몰을 구축한다.

 프로젝트기능 :

 5. 주문 목록 페이지 : 주문 목록을 볼 수 있다.
 6. 주문 상세 페이지 : 주문 상세 정보를 볼 수 있다.
 7. 회원가입 페이지 : 회원가입을 할 수 있다.
 8. 로그인 페이지 : 로그인을 할 수 있다.
 9. 로그아웃 페이지 : 로그아웃을 할 수 있다.
 10. 회원정보 수정 페이지 : 회원정보를 수정할 수 있다.
 11. 회원탈퇴 페이지 : 회원탈퇴를 할 수 있다.
